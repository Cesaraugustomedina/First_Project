# Superstore sales


«Superstore Sales» es una recopilación completa y versátil que ofrece información valiosa sobre las ventas, el comportamiento de los clientes y el rendimiento de los productos. Este conjunto de datos constituye un valioso recurso para realizar análisis en profundidad.

El conjunto de datos, que contiene información procedente de diversas regiones y segmentos, permite analizar tendencias, patrones y correlaciones en las ventas y las preferencias de los clientes. El conjunto de datos incluye transacciones de venta, lo que permite a los investigadores y analistas comprender los patrones de compra, identificar los productos con mayor demanda y evaluar la eficacia de los distintos métodos de envío.

## Acerca del conjunto de datos:

| Nombre de variable| Tipo de dato | Descripción de variable |
| :--- | :---: | ---: |
|Row ID | Decimal (PM)| Es el identificador único de cada fila |
| Order ID | Varchar(30)| ID de la orden de venta|
| Order Date | Date | Fecha de la orden de venta |
| Ship Date | Date | Fecha de envío |
| Ship Mode | Varchar(30) | Categoria del envío  |
| Customer ID | Varchar(25) | ID del cliente |
| Customer Name | Varchar(20) | Nombre del cliente |
| Segment | Varchar(25) | Segmento al que pertenece el cliente |
| Country | Varchar(25) | País de la venta |
| City | Varchar(25) | Ciudad de la venta |
| State | Varchar(25) | Estado de la venta |
| Postal Code | Varchar(15) | Codigo postal asoaciado a la venta |
| Region | Varchar(15) | Region asociada a la venta |
| Product ID| Varchar(30) | ID del producto asociado a la venta |
| Category | Varchar(25) | Categoria asociado al producto de la venta |
| Sub-category | Varchar(25) | Subcategoría asociada al producto de la venta |
| Product Name | Varchar(20) | Nombre del producto asociado a la venta |
| Sales | Float | Precio de la venta |
| Quantity | Decimal | Cantidad vendida |
| Discount | Float | Descuento asociado a la venta|
| Profit | Float | Ganancias derivadas de la venta |
